# Question: (Easy)

Given two binary trees, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical and the nodes have the same value

## Example 1:

>Input:		1        1   <br>
>		   / \      / \  <br>
>		  2   3    3   2 <br>
>
>        [1,2,3],  [1,3,2] <br>
>
>Output: true

## Example 2:

>Input:		1      1   <br>
>		   /        \  <br>
>		  2          2 <br>
>
>        [1,2],  [1,null,2] <br>
>
>Output: false

## Example 3:

>Input:		1        1   <br>
>		   / \      / \  <br>
>		  2   1    1   2 <br>
>
>        [1,2,1],  [1,1,2] <br>
>
>Output: false
